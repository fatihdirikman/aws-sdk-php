This guide focuses on the AWS SDK for PHP interface to |service_name|. This guide assumes that you have already downloaded
and installed the AWS SDK for PHP 2. See :doc:`installation` for more information on getting started.

Creating a client
-----------------

The first thing you need to do is create a client object. There are a couple of ways to do this using the SDK.

Factory method
~~~~~~~~~~~~~~

The easiest way to get up and running is to use the factory method. Simply pass in your access keys and a region to
connect to.
